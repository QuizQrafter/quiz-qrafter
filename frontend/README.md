# frontend

## Getting Started

### Prerequisite(s)

- [Node.js](https://nodejs.org/)

### Run

#### 1. Set environment variables

Copy the contents of the `.env.example` file to a new `.env` file

#### 3. Install dependencies

```sh
$ npm install
```

#### 4. Run application

```sh
$ npm run start
```
